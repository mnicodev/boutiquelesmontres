Allow your users to zoom in on the images of their products with this powerful zoom tool designed for high quality photos.

Zoom Product Pro helps visitors to observe in detail their products with a high degree of zoom. Specially designed to show details in high quality photographs. Increase conversions of your Prestashop store allowing your customers to see the finish and details of each of your products.

Settings

In module configuration are configured parameters with that is made the zoom to images of products.

Identifier image

This refers to the identifier of the large image displayed on the product in the store. If you do not know what this is, please do not touch it and leave the default option that comes pre- configured for 99% of existing stores. Otherwise, you can modify to your needs.

Width

This refers to the width of the frame where the image is displayed with Zoom. The minimum size is 600px but the maximum can be set as you need. It is always recommended that the maximum width is that of his store so that users do not have trouble navigating.

Height

This refers to the height of the frame where the image is displayed with Zoom. The minimum size is 600px but the maximum can be set as you need.

Postfix of the name images

Don't modify this field without our permission.

Show gallery image

Turning this feature is displayed vertically to the right of the image that you can zoom, a list of all images in the gallery. Being able to move forward through these thumbnails.

Full screen

Turning this feature, the box showing the picture to the gallery, occupies the entire width and height of the browser window of the visitor

Are you using watermark yours images?

Enable this option if you are using watermark on images.


ENABLE WATERMARK ON IMAGES.

In ZoomProductPro module configuration, you have the option of using watermark in your photos. The process is managed by the engine Prestashop and guide you step by step how to activate this feature to integrate the module ZoomProductPro.

Steps:

  1. Go to the module configuration.
  2. Enable option Are you using watermark yours images?
  3. Go to Preferences > images in Prestashop
  4. Click on Add new image type
  5. On textbox Name for the image type enter zoomproductpro (without spaces and all lowercase)
  6. In the Width and Height fields define the dimensions of the image used as watermark.
  7. Enable option called Products
  8. Save the changes.
  9. Go now to the water maker module of Prestashop, on Modules > Category Management > Watermark
  10. Activate the Module and click on configure.
  11. Select the image to use as a watermark, select the location, the percentage of transparency and check the option called zoomproductpro
  12. Save the settings.
  13. Go back to the tab Preferences > Images on Prestashop
  14. Although we have set the watermark, it begins to work with the photos I uploaded from now on, so you have to force Prestashop to update existing images with new images with a watermark. Go to the bottom of this page, search the section of Regenerate Thumbnails and in the first dropdown list select Products and in the second dropdown list select zoomproductpro.
  15. Click on Regenerate thumbnails

Done!, that's all. Now enjoy the watermark with your logo on all images of your store

Enable 360

Enable option so that customers can see the product images in 360º.

Where do i save the product images for the slider 360?.
  1. Go to the zoomproductpro_360 folder, at the root of shop.
  2. Create a folder with the product id.
  3. Paste images into folder. (The name of the images must be as follows: 01, 02, ..., 09, 10, 11, ...n )

Extension of images

Enter extension of the images to slider 360.
